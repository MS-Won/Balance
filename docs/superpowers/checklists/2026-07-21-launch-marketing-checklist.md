# 오늘의 밸런스 게임 — 런칭 홍보 실행 체크리스트 (2026-07-21)

> 코드 아님, 운영 체크리스트. 참고 스펙: `docs/superpowers/specs/2026-07-15-growth-marketing-design.md`
> 전제: MVP 테스트 완료, URL 홍보 예정. 카카오톡 채널 미개설, 커뮤니티 활동 계정 없음(신규 시작), 인스타그램 계정은 이미 보유.

이 전제(계정/채널이 전혀 없는 상태) 때문에 스펙의 Track A(커뮤니티 집중 발사)를 지금 바로 실행하면 신규 계정 도배 신고로 계정이 먼저 죽는다. 그래서 아래는 스펙 순서 그대로가 아니라, **신뢰도 쌓기 → 소프트 런칭(인스타/카카오) → 커뮤니티 발사** 순으로 재배치했다.

## Phase 0 — 이번 주: 인프라 + 계정 시딩

- [ ] **카카오톡 채널 개설** (카카오톡 채널 관리자센터, 무료, 사업자등록 불필요) → 채널 ID 확보
- [ ] `NEXT_PUBLIC_KAKAO_CHANNEL_ID`를 `.env.local`과 Vercel 프로젝트(`balance-game-mvp`) 환경변수 양쪽에 설정 → 재배포해서 `KakaoChannelButton`이 실제로 렌더링되는지 확인
- [ ] 배포된 실제 URL 확정 (Vercel 프로덕션 도메인) — 이후 모든 공유/발사에 이 URL 사용
- [ ] **source 파라미터 규칙 확정**: `?src=fmkorea`, `?src=insta_story`, `?src=kakao` 등. 발사할 때마다 반드시 붙이기로 스스로 룰 고정 (스펙 "측정 루프")
- [ ] 커뮤니티 계정 신규 가입 시작 — **홍보 목적이라는 티 내지 말고 일반 활동으로 시작** (에펨코리아/디시/네이트판 등은 등업 조건·신규계정 도배 감지가 있어서, 가입 직후 홍보 글부터 올리면 삭제·차단 위험이 큼). 이번 주엔 그냥 일반 댓글/눈팅으로 계정을 자연스럽게 만들어두는 것만 목표.

## Phase 1 — 다음 2주: 소프트 런칭 (인스타 + 카카오 위주)

커뮤니티 계정은 아직 신뢰도가 없으니, 이 기간은 **이미 가진 채널(인스타그램)** 로만 운영한다.

- [ ] 인스타그램: 매일 그날의 밸런스 게임 주제를 **스토리 투표 스티커**로 동시 게시 (사이트 링크는 스티커 링크로, source 파라미터 포함). 이건 콘텐츠 검증(어떤 주제가 반응 좋은지)과 실유입을 동시에 얻는 용도.
- [ ] 주 2~3회는 스토리 대신 **피드/카드뉴스**로 "역대 반응 좋았던 밸런스 게임" 요약 올려서 저장/공유 유도
- [ ] 카카오톡 채널: 아직 친구가 없으므로 이 기간은 **인스타 바이오/스토리에 채널 추가 유도** 위주로 친구 수를 먼저 쌓는다 (친구톡 발송은 최소 친구가 있어야 의미 있음)
- [ ] 매일 admin에 게임 등록하는 루틴에 "인스타 스토리 게시"를 같이 묶어서 습관화 (같은 리듬)
- [ ] source 파라미터 기준으로 어떤 콘텐츠/주제가 클릭·채팅 참여로 이어지는지 주 단위로 확인

**이 단계의 목표는 트래픽 규모가 아니라 (1) 카카오 채널 친구 수 확보, (2) 어떤 주제가 바이럴성 있는지 데이터 확보, (3) 커뮤니티 계정 신뢰도 확보 3가지를 동시에 진행하는 것.**

## Phase 2 — 신뢰도 확보 후: 커뮤니티 집중 발사 (Track A 본격 실행)

커뮤니티별 등업 조건 충족 + 일반 활동 이력이 좀 쌓인 뒤 시작 (커뮤니티마다 다르지만 대략 2~4주 활동 후가 안전).

- [ ] 타겟: 에펨코리아 유머게시판, 디시인사이드 관련 갤러리, 네이트판/여성시대, 인스타·스레드 밈/밸런스게임 계정
- [ ] 빈도: **주 1~2회**, 매번 Phase 1에서 검증된 "가장 반응 좋았던/자극적인" 주제 선택
- [ ] 로테이션: 같은 커뮤니티엔 주 1회 이하
- [ ] 타이밍: 저녁 8~11시 (모바일 트래픽 몰리는 시간), admin에 게임 등록하는 것과 같은 리듬
- [ ] 광고 예산 원칙 — **선검증 후증폭**: 미리 광고 태우지 않고, 유기적으로 반응 좋았던 포스트만 그때 소액으로 인스타/스레드 부스트
- [ ] 이때부터 카카오톡 채널 친구톡도 매일 admin 게임 등록과 같은 루틴으로 수동 발송 시작 ("오늘의 밸런스 떴어요! [링크]")

## 범위 밖 (스펙 기준, 이번 체크리스트에도 동일 적용)

나노인플루언서 협업, 알림톡 자동화(API 연동), 유료 배너 광고, 트래픽이 실제로 쌓인 이후의 확장 전략.
