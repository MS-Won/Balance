# 오늘의 밸런스 게임 — 디자인 스펙 (v1)

메인 화면(`/`) 전면 리디자인. 로직/데이터는 그대로, 표현(스타일·레이아웃·폰트·카피)만 바꾼다.
관리자 페이지는 이 스펙에 필요한 범위(선택지별 상세가정 입력)만 변경한다.

레퍼런스 목업: `docs/design-mockup.html` (구현 시 픽셀·톤의 기준).

## 1. 컨셉 — "진지한 얼굴로 헛소리"

UI 자체는 차분하고 세련되게(HydroTrack 무드), 웃음은 **타이포와 카피**가 담당한다.
반듯한 틀 위에서 굵은 폰트가 헛소리를 진지하게 말하는 **대비 = 데드팬/부조리 개그**.
화면을 요란하게 꾸미지 않는 것이 핵심.

## 2. 아트 디렉션 / 디자인 토큰

- 라이트 전용 (기존 `prefers-color-scheme` 다크 블록 제거)
- 밝은 오프화이트 배경 + 흰 둥근 카드 + 부드러운 그림자 + 넉넉한 여백
- A vs B 대비 2색: **A = 코랄(따뜻, 왼쪽), B = 민트(차가움, 오른쪽)**

| 토큰 | 값 |
|---|---|
| `--bg` | `#F5F4EF` (따뜻한 오프화이트) |
| `--card` | `#FFFFFF` |
| `--ink` | `#1E2024` |
| `--muted` | `#8B9098` |
| `--line` | `#EDEBE4` |
| `--coral` (A) | `#FF6B54` |
| `--coral-soft` | `#FFE7E1` |
| `--coral-deep` (선택 테두리) | `#B8341F` |
| `--mint` (B) | `#18BE9F` |
| `--mint-soft` | `#D6F3EC` |
| `--mint-deep` (선택 테두리) | `#0B8A72` |
| `--radius` | `22px` (카드), pill/버튼은 13~20px |
| `--shadow` | `0 10px 28px rgba(31,32,36,.07)` |
| `--shadow-sm` | `0 4px 14px rgba(31,32,36,.06)` |

헤더는 코랄 그라디언트(`135deg, #FF8A73 → #FF6B54 → #F4553D`) 한 곳만 절제해서 사용.
마이크로 인터랙션(막대 애니메이션, 숫자, 버튼 눌림, ㅇㅈ 토글)은 가볍게.

## 3. 타이포 시스템 (next/font/google, 셀프호스팅)

| 역할 | 폰트 |
|---|---|
| 선택지 A/B 라벨, %숫자, 카운트다운, ㅇㅈ 워드아트 | **Black Han Sans** (초굵은 간판체) |
| 질문 + 선택지 상세가정 + 앱 이름 + 섹션 라벨 | **Jua** (둥근 능청) |
| 본문·닉네임·채팅·UI·기타 | 시스템 산세리프(현 기본 유지) |

- 긴 질문/선택지의 줄바꿈은 CSS `text-wrap:balance`로 자동 균형 처리 → **관리자는 줄바꿈을 신경 쓰지 않고 한 줄로 입력**하면 됨(수동 `<br>` 불필요).

## 4. 메인 `/` 레이아웃 (위 → 아래)

1. **헤더** — 앱 이름 "오늘의 밸런스"(Jua) + 우측 "다음 밸런스까지 HH:MM:SS"(라벨 작게 + 시간 Black Han Sans). 코랄 그라디언트 카드.
2. **VERSUS 블록** (하나의 카드, `overflow:hidden`):
   - 상단 작은 회색 문구: **"지금까지 N명 참여했어요"** (총 투표수)
   - **질문**: `Q. ` 접두어 + 질문(Jua, 중앙, `text-wrap:balance`)
   - **좌/우 선택지 패널 2개** (`min-height≈230px`, A 코랄톤 왼쪽 / B 민트톤 오른쪽, 3px 간격):
     - 각 패널 = **선택지 라벨**(Black Han Sans, 최대 크기) + **그 선택지의 상세가정**(Jua, 괄호, 작게)
     - **패널 자체가 투표 버튼** (별도 투표 버튼 없음)
     - **선택됨 표시 = 색 반전 + 테두리**: 고른 쪽은 꽉 찬 진영색 + 흰 글씨 + 안쪽 5px 딥컬러 테두리 링 + 우상단 ✓ 뱃지. 안 고른 쪽은 옅은 톤 유지.
   - **투표 현황 막대(풀블리드)**: 패널 바로 아래, 카드와 **동일 가로폭(주변 여백 0)**. 코랄/민트 비율 채움. **%숫자를 막대 위에 올려 A는 좌측정렬·B는 우측정렬**(흰 Black Han Sans). 카드 하단 모서리 라운드에 맞춰 클립.
3. **대표 의견** — A/B 각 진영 최다 ㅇㅈ 의견 카드 2개(코랄톤/민트톤). **"대표 의견" 글자 없음**; 그 자리에 **닉네임 + `ㅇㅈ N`**(진영색, 굵게), 아래 의견 본문.
4. **채팅**:
   - 섹션 라벨 **"의견을 써주세요 :)"**(Jua) + 우측 정렬 토글 **"ㅇㅈ순 ▾"**
   - 메시지 = 진영색 점 + 흰 말풍선(닉네임·본문) + **오른쪽에 ㅇㅈ 버튼**(작게)
   - 입력창(투표+닉네임 후 활성) + 전송
5. **명예의 전당** — top10, 메달(🥇🥈🥉) + 닉네임 + 질문 + `ㅇㅈ N` 칩.
6. **닉네임 프롬프트** — 모달, 리스타일.

> **"어제의 결과"(YesterdayResult)와 "광고 자리"는 메인에서 제거.** (광고는 Phase 2에서 재도입)

## 5. ㅇㅈ 브랜딩 (인정 = ㅇㅈ)

기존 👍(따봉)은 **전면 제거**. 인정 지표는 어디서나 **"ㅇㅈ" 워드아트 글리프**로 통일:
- Black Han Sans, 살짝 기울임(`rotate(-5deg)`), 하나의 이모지처럼 인라인 사용.
- 채팅 버튼: 안 누름 = 흰 버튼/코랄 ㅇㅈ, 누름 = 코랄 채움/흰 ㅇㅈ (작은 크기).
- 정렬 라벨 "ㅇㅈ순", 대표의견·명예전당 카운트 앞의 ㅇㅈ 등.

## 6. 카피 톤 (데드팬/부조리)

로딩·빈 상태·라벨을 능청스럽게. 예:
- 로딩 "우주의 균형을 재는 중…"
- 게임 없음 "오늘은 세상이 평화롭습니다. (문제가 없다는 뜻)"
- 투표 0 "아직 아무도 용기 내지 않았다."
- 참여 문구 "지금까지 N명 참여했어요"

실제 문구는 구현 때 소수 후보로 다듬는다. 과하지 않게, 한두 스푼.

## 7. 관리자 / DB 변경

- **`question`** 컬럼: 이미 적용됨(마이그레이션 0005).
- **선택지별 상세가정**: 게임 단위 `description`(질문의 상세가정)을 폐기하고,
  **`choice_a_description`, `choice_b_description`(둘 다 nullable text)** 신규 추가(마이그레이션 0006).
  - 관리자 폼: `날짜 / 질문 / 선택지A + 상세가정A(선택) / 선택지B + 상세가정B(선택)`
  - `createGame`이 두 컬럼을 저장, 메인은 각 패널에 해당 상세가정 표시.
  - 기존 `description` 컬럼은 미사용(남겨두되 UI에서 참조 안 함).

## 8. 범위 / 비범위

- **범위**: 메인 `/` 전면 리스타일, 폰트 도입, 디자인 토큰, ㅇㅈ 브랜딩, 선택지별 상세가정(마이그레이션+admin+메인), 카피 일부, `layout.tsx` 메타데이터 교체.
- **비범위**: 투표/채팅/인정/롤오버 등 **로직·데이터 흐름 불변**. `/admin`은 위 폼 변경 외 전면 리디자인 안 함.

## 9. 구현 노트

- `layout.tsx`: `next/font/google`로 `Black_Han_Sans`(weight 400), `Jua`(weight 400) 로드 → CSS 변수(`--font-poster`, `--font-hand`)로 노출. `metadata` 제목/설명 교체.
- `globals.css`: 위 토큰 정의(`:root`), 다크 블록 제거, 목업의 컴포넌트 클래스(versus/col/tally/oj/endorse/rep/hof 등) 이식.
- 컴포넌트 매핑(대략):
  - 신규 `BalanceCard`(질문+좌우 선택지 버튼+투표바) ← 기존 `VoteGraph` + 투표 버튼 대체
  - `Header`, `RepresentativeOpinionBar`, `ChatFeed`, `ChatInput`, `HallOfFame`, `NicknamePrompt` 리스타일
  - `page.tsx`에서 `YesterdayResult`·`AdPlaceholder` 제거
- `useActiveGame`/`useVotes` 등 훅 시그니처는 유지(가능하면). 참여수 N = 현재 게임 총 투표수(tally.total).
- 정적 게이트: `npm run lint` / `npx tsc --noEmit` / `npm run test` 그린 유지.
